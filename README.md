# lendsqr-fe-test

A responsive React/Vite implementation of the Lendsqr frontend assessment with:

- Login, Dashboard, Users, and User Details pages
- Mock API with 500 generated user records
- Server-style filtering, pagination, and sorting
- Local storage-backed user detail persistence

## Public links

- Live app: https://wahmed3900-lendsqr-fe-test.vercel.app
- Public GitHub repository: https://github.com/wahmed3900/lendsqr-fe-test

## What was completed

- Built a mobile-responsive React/Vite SPA with protected routes for Login, Dashboard, Users, and User Details.
- Implemented a mock API that generates 500 users and supports search, status filtering, sorting, and server-style pagination.
- Added local storage-backed persistence for user notes/details on the User Details page.
- Verified behavior with Vitest tests and a production build.
- Deployed the app to Vercel and published the source code to a public GitHub repository.
